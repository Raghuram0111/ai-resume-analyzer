# AI Resume Analyzer

## 📌 Description
This project analyzes resumes and compares them with job descriptions using Python and NLP techniques.

## 🚀 Features
- Resume PDF upload
- Skill matching
- Score calculation
- Suggestions for improvement

## 🛠️ Technologies Used
- Python
- Flask
- HTML, CSS
- NLP
